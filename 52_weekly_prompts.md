## Usage Instructions

Each prompt is designed to be used as follows:

1. Copy the weekly prompt
2. Submit it to generate a detailed implementation plan
3. The plan should include:
   - Feature breakdown with user stories
   - Technical architecture using TypeScript/Vite/React
   - Component structure and data flow
   - Step-by-step implementation guide
   - Code examples for key functionality
   - Testing considerations
   - Deployment recommendations

All projects emphasize:
- Local-first data privacy
- Professional-grade UI/UX
- Meaningful, domain-specific features
- Practical utility for the target profession
